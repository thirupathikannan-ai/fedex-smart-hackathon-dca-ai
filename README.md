# FedEx Smart Hackathon – AI-Powered DCA Management Platform

## Overview
This project proposes an AI-driven, enterprise-ready platform to modernize Debt Collection Agency (DCA) management at FedEx.

## Problem Statement
FedEx manages overdue customer accounts through multiple DCAs using manual and fragmented processes, leading to delays, weak governance, and poor predictability.

## Solution
A centralized platform integrating:
- AI-based recovery prediction
- Intelligent case prioritization
- SLA breach prediction
- DCA performance scoring
- Secure dashboards and governance

## System Architecture
FedEx Portal → Workflow Engine → AI/ML Layer → DCA Portal → Analytics Dashboard

## AI Models Used
1. Recovery Probability Prediction
2. Case Prioritization Engine
3. SLA Breach Prediction
4. DCA Performance Scoring
5. Escalation Risk Detection

## Business Impact
- Faster recoveries
- Reduced overdue ageing
- Improved predictability
- Enterprise-grade governance
